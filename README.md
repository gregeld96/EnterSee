## ENTERSEE
> This is an app to see a list of movie where the user could do CRUD on it, this app was created using Apollo GrapQL technology where React was used as the front end framework and MongoDB was the RDMS.

- Backend Technology:
  - Node JS
  - MongoDB
  - Redis
  - GraphQL
  - Apollo Server

- Frontend Technology:
  - React JS
  - Redux
  - Apollo Client
